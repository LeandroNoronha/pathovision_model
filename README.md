# research_project-unisinos

This repository is dedicated to the development of my final paper at Unisinos.  
It includes all the code implementations and written content, structured in LaTeX while adhering to ABNT standards.  

The repository serves as a comprehensive workspace for organizing research, experiments, and documentation, ensuring consistency and proper formatting throughout the project.

## Versioning

The evolution of this project has generated different versions of the classifier:

- **v4.0.0 (Stable)**  
  - Better legibility.  
  - Improved acquisition of metrics.  
  - Recommended version for use.

- **v3.0.0 (Stable_old)**  
  - First stable and functional version of the model.  
  - Trained with EfficientNetB2 and validated metrics.

- **v2.0.0 (Unstable)**  
  - Contains compatibility issues with Keras/TensorFlow.  
  - Not recommended for production use.

- **v1.0.0 (Broken/Deprecated)**  
  - Initial prototype.  
  - Does not work with current dependency versions.
